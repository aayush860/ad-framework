// 02 — CORE INNOVATION

Zero-Code API Engine

The framework's most powerful feature: a declarative JSON contract that auto-generates fully callable Python methods at runtime — with zero file generation and zero extra disk space.

Runtime Dynamic Dispatch Engine
Adding a new API endpoint requires only a JSON entry — no Python code, no generated files, no rebuilds.
STEP 01 — DECLARE
JSON Contract
Define endpoint, URI, HTTP verb, expected status code in apis.json
"siem_association": { "URI": "crux/v3/...", "POST": {...} }
STEP 02 — INTERCEPT
__getattr__ Fires
Python can't find the method — __getattr__ intercepts the attribute lookup
.post_siem_association → __getattr__("post_siem_association")
STEP 03 — PARSE
Name → HTTP Verb
Method name is split to extract HTTP verb and endpoint key
"post_siem_association" → verb="POST" + key="siem_association"
STEP 04 — EXECUTE
Callable Returned
A configured callable is returned and invoked with user-supplied kwargs
POST https://host/crux/v3/mgmt-pop/siem {body}
generate_apis.py — Core Engine (30 lines)
import json, requests

class Rest:
    def __init__(self, host):
        self.host = host
        self.config = {}

    def generate_apis(self, json_path):
        # Load declarative contract — drives everything
        with open(json_path) as f:
            self.config = json.load(f)

    def __getattr__(self, group_name):
        # Fires when .eaa_metrics is accessed — not a real attribute
        if group_name in self.config:
            return ApiGroup(self.config[group_name], self.host)
        raise AttributeError(f"No group '{group_name}'")


class ApiGroup:
    def __init__(self, config, host):
        self.config = config
        self.host = host

    def __getattr__(self, method_name):
        # "post_siem_association" → POST + siem_association
        verb, endpoint_key = method_name.split("_", 1)
        uri = self.config[endpoint_key]["URI"]

        def make_request(**kwargs):
            return requests.request(
                method=verb.upper(),
                url=f"https://{self.host}/{uri}",
                json=kwargs.get("BODY", {}),
                params=kwargs.get("PARAMS", {})
            )
        return make_request  # returns callable, not result

// 03 — DESIGN PATTERNS

Patterns in Practice

Every pattern in the codebase serves a specific purpose — not applied for academic reasons but to solve real problems of scale, maintainability, and test isolation.

🎭
Dynamic Dispatch
generate_apis.py → __getattr__
Intercepts any method call at runtime. Converts method names like post_siem_association into configured HTTP callables from JSON config.
🏗️
Facade
GenericUtils → check_and_associate_siem()
Hides the complexity of check → delete → create SIEM flow behind a single method. Test layer never knows about 3 underlying API calls.
🔀
Strategy
DatadogMetrics / CloudWatchMetricFetcher
Both SIEM utilities share the same interface contract — init with name → decrypt config → get agent ID → query. Swappable at runtime via SIEM_MAP.
🗄️
Flyweight + Cache
PortalRest → __session_auth_map
Class-level token cache keyed by (identity, account, contract). Tokens reused across test instances. Proactively refreshed at 50 min before 60-min expiry.
📋
Declarative Config
apis.json / SIEM_MAP
Behaviour defined in data, not code. Adding an API or SIEM type requires zero Python changes — only config updates.
🏭
Factory (Closure)
make_request() in ApiGroup
__getattr__ returns a configured function (closure), not a result. The callable captures host, verb, and URI in its scope — produced on demand.
🔧
Template Method
PortalRest.__init__
Defines the skeleton of auth initialization — validate → cache check → create if needed → assign. Each auth strategy fills in specifics.
🧹
RAII / Destructor
ResourceSpiker.__del__()
SSH connection lifecycle tied to object lifetime. __del__ ensures cleanup even if test fails. Noted: context manager would be more deterministic.

from cryptography.fernet import Fernet

class SecretsCryptography:
    def __init__(self, key: str, file_for_decryption: str):
        self.cipher = Fernet(key.encode())

    def decrypt_file_to_dict(self) -> dict:
        # Encrypted .enc file → Python dict at runtime
        with open(self.file_for_decryption, "rb") as f:
            token = f.read()
        return json.loads(self.cipher.decrypt(token))
